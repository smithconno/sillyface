Scale Essay

The definition for a scale is a function that maps an input domain to an output range. It is a mathematical relationship between the data and the visual, whatever that may be. It is basically mathematically solving the input for an output. You can refine the plot by changing the yScale form. You will then need to incorporate padding to push the elements in. You can make time scales with d3 algorithms. Since javascript understands time, you just need to create a d3 function.
